## 🔧 技术规格

### 支持的数据库
- PostgreSQL（含 pgvector 扩展）
- MySQL
- Qdrant（向量数据库）

### Go 版本要求
- **Go 1.21+**（支持 Fuzz 测试）

### 核心特性
1. **SQL Builder** - 关系数据库查询构建
2. **Vector DB** - Qdrant/pgvector 客户端
3. **Hybrid Query** - 向量搜索 + 关系过滤
4. **Auto Filtering** - 自动过滤 nil/0/空值
5. **Type Safe** - 编译时类型检查
