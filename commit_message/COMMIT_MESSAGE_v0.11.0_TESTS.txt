### 2. Fuzz 测试

**新增文件**: `fuzz_test.go`

**4 个 Fuzz 函数**:

#### FuzzStringConditions
- 测试字符串条件：Eq, Ne, Like, LikeLeft
- 种子语料包含：
  - 空字符串
  - SQL 注入: `'; DROP TABLE users; --`
  - 超长字符串
  - 正常业务字符串

#### FuzzNumericConditions
- 测试数值条件：Gt, Gte, Lt, Lte
- 种子语料包含：
  - 零值
  - 负数
  - int64 最大值: 9223372036854775807
  - 正常业务数值

#### FuzzPagination
- 测试分页：Limit, Offset
- 种子语料包含：
  - 正常分页: (10, 0), (100, 50)
  - 边界值: (0, 0), (-1, -1)
  - 大数值: (1000000, 999999)

#### FuzzXCondition
- 测试硬编码条件：X()
- 种子语料包含：
  - 标准条件: `id > ?`
  - 复杂条件: `BETWEEN ? AND ?`
  - 空字符串
  - 各种 SQL 片段
